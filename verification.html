FOLLOW THESE EASY STEPS

TO CHECK AUTHENTICITY CODE

    Locate authentication label to find security code. ( Old sticker will be discontinue starting from Feb 2019 onward )

  • Scratch & Check

    Scratch and check for ‘10 character security code’ on both the 1st line and 2nd line of the authenticity sticker.

  • Scan QR Code

    Scan QR Code or go to
    'http://www.vulkanpharma.com/verification.html' then click 'Check Code'

  • Enter Code

    Enter the exact code display on the 1st and 2nd line respectively. Click 'CHECK' to check your code.
